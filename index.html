Scrabble Game

Instructions

Welcome to the Scrabble Game! Here's how to play:

  • Drag tiles from the tile rack to the Scrabble board to form words.
  • Words must be connected, and tiles must be placed adjacent to one another without gaps.
  • Click the Validate Word button to check if your word is valid and calculate your score.
  • Refresh the tile rack using the Refresh Letters button for a new set of tiles.
  • If you want to clear the board, use the Clear Board button to move all tiles back to the rack.
  • Start a new game anytime with the Start Over button.
  • The game keeps track of your highest score.

Rules

  • Words must contain at least two letters to be valid.
  • Each tile has a point value, and bonus squares on the board can multiply your score.
  • Double-word score squares multiply the total word score by 2.
  • Words can only be validated once; repeated validation will not earn additional points.
  • The game uses a standard English dictionary to validate words.
  • Blank tiles can be used as any letter, but they have no point value.
